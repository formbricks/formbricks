---
title: "Teams & Roles (RBAC)"
description: "Granularly control which users have access to specific Workspaces and surveys."
icon: "user"
---

Each User can be a member of any number of Teams.

Each Team can have Read, Write or Manage access to any number of Workspaces.

In the Community Edition, each User has Admin rights.

In the Enterprise Edition, you can granularly assign Roles and Memberships to users.

Here are [all details of user management](https://formbricks.com/docs/platform/features/user-management#memberships)
